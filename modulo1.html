Grade curricular: 1º módulo


QUALIFICAÇÃO PROFISSIONAL TÉCNICA DE NÍVEL MÉDIO DE AUXILIAR EM DESENVOLVIMENTO DE SISTEMAS


Disciplinas ministradas no primeiro módulo do curso:

Linguagem, Trabalho e Tecnologia (LTT)

LTT se trata de como utilizar a linguagem, tendo como base principal a língua portuguesa. LTT ensina como analisar e desenvolver textos administrativos ou técnicos, como se comunicar utilizando do vocabulário técnico e como interpretar a linguagem técnico-científica.

Programação e Algoritmos (PA)

PA se trata da lógica, que é a base de todo o curso. PA ensina como os alunos devem utilizar da lógica para resolver problemas, ensinando também as maneiras de utilizar a lógica por meio das linguagens de programação, como Java, por exemplo.

Banco de Dados I (BD I)

Um Banco de Dados é um conjunto de dados relacionados entre si, que ficarão organizados de modo a fornecer informações. Em BD I, os alunos aprendem o mais básico sobre os Bancos de Dados, como o conceito e a modelagem inicial de um.

Análise e Projeto de Sistemas (APS)

APS se trata de projetos de sistemas. APS se encarrega de entregar a capacidade aos alunos de modelar projetos de sistemas e utilizar estes modelos para desenvolver sistemas, fazendo com que os mesmos desenvolvam as habilidades de utilizar métodos de abordagem, coleta de dados e procedimentos de pesquisa, aplicando tais conceitos em modelos ágeis e projetos de software.

Design Digital (DD)

DD se trata de desenvolver interfaces visuais para softwares e sites, os tornando além de belos, também funcionais. DD procura a melhor solução possível de acordo com as limitações do meio que será publicado, podendo, muitas vezes, superar esses desafios e apresentar novas propostas.

Programação Web I (PW I)

PW I se trata da criação e formatação de sites. PW I tem o objetivo de capacitar os alunos a ter a habilidade de desenvolver e implementar páginas para a internet, utilizando-se de linguagens de marcação de texto, linguagens de script e construção de folhas de estilo.

Fundamentos da Informática (FI)

FI se trata basicamente do que é necessário para que um computador funcione, mostrando desde o funcionamento dos processadores até o funcionamento do sistema operacional. Em FI os alunos estudam como um Sistema Operacional funciona, como um computador funciona, o que é o hardware, o que são sistemas de arquivos e estudam também a história dos computadores.

Técnicas de Programação (TP)

TP se trata de técnicas para facilitar a vida do aluno como programador. Em TP os alunos aprendem o que é a programação, técnicas que melhoram o código e a como deixar o código limpo e organizado.
