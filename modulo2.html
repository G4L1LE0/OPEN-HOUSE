Grade curricular: 2º módulo


QUALIFICAÇÃO TÉCNICA DE NÍVEL MÉDIO DE AUXILIAR EM DESENVOLVIMENTO DE SISTEMAS


Disciplinas ministradas no primeiro módulo do curso:

Inglês Instrumental (II)

O II tem como base a língua inglesa. O II tem como objetivo transpassar a língua inglesa aos alunos, fazendo com que eles tenham a capacidade de analisar e produzir textos na mesma.

Desenvolvimento de Sistemas (DS)

DS se trata de iniciar a projeção dos sistemas, selecionando linguagens e ambientes para o desenvolvimento dos mesmos. DS tem o objetivo de capacitar os alunos a projetar sistemas, aprender os conceitos das linguagens de programação orientadas a objeto, criar programas orientados a objetos e conectar o programa a um banco de dados.

Banco de Dados II (BDII)

BDII se trata de implementar os bancos de dados modelados inicialmente num banco de dados físico. BDII tem o objetivo de introduzir os alunos a linguagem SQL (Structured Query Language em inglês, ou Linguagem de Consulta Estruturada, em português) que é a linguagem padrão utilizada para manipular dados em bancos de dados relacionais.

Internet e Protocolos (IP)

IP se trata de como funciona a internet. Em IP aprendemos como funciona a internet por debaixo dos panos e também a projetar desde redes locais domésticas até usar tais técnicas a nosso favor no desenvolvimento de aplicações.

Programação Mobile I (PMI)

PMI se trata de criação de aplicativos mobile. Em PMI os alunos aprendem a criar, como o nome já diz, aplicativos para dispositivos móveis, como os aplicativos para celular que são instalados pela Play Store, Apple Store e afins. São criados desde jogos até aplicativos para bancos, por exemplo.

Programação Web II (PWII)

PWII se trata basicamente da implementação de bancos de dados aos sites. PWII tem como objetivo ensinar aos alunos como implementar bancos de dados aos sites, utilizando principalmente a linguagem PHP.

Planejamento do Trabalho de Conclusão de Curso (TCC) em Desenvolvimento de Sistemas (TCCI)

TCCI se trata de iniciar os projetos de TCC dos alunos. TCCI tem o objetivo de fazer os alunos começarem a colocar em prática tudo o que foi e será aprendido no decorrer do curso. É nessa fase onde os alunos dão início, também, à organização de grupos e ideias para a composição dos seus trabalhos de conclusao de curso.
